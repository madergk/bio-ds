# Content

El Bio Design System proporciona estilos base para contenido, incluyendo tipografía, imágenes, tablas y más. Estos estilos están diseñados para proporcionar una base consistente y accesible para todo el contenido.

## Componentes de Content

### [Reboot](./reboot.md)
Reboot es una colección de estilos base que normaliza y corrige inconsistencias entre navegadores. Proporciona una base sólida y consistente para construir sobre ella.

### [Typography](./typography.md)
El sistema de tipografía del Bio Design System utiliza la fuente Sora y proporciona una jerarquía clara de tamaños, pesos y alturas de línea.

### [Images](./images.md)
Estilos y utilidades para imágenes responsivas, incluyendo clases para hacer imágenes adaptables y estilos para diferentes tipos de imágenes.

### [Tables](./tables.md)
Estilos para tablas, incluyendo variantes, estados y utilidades para crear tablas responsivas y accesibles.

### [Figures](./figures.md)
Estilos para elementos figure, comúnmente usados para imágenes con captions y contenido relacionado.

## Conceptos Clave

### Normalización Cross-Browser

El Bio Design System utiliza Reboot para normalizar estilos entre navegadores, proporcionando una base consistente.

### Tipografía Sora

El sistema utiliza **Sora** como fuente principal, proporcionando una experiencia de lectura clara y moderna.

### Responsive por Defecto

Muchos elementos de contenido son responsivos por defecto, adaptándose automáticamente a diferentes tamaños de pantalla.

## Ejemplo Rápido

```html
<!-- Typography -->
<h1>Título Principal</h1>
<p>Texto base con fuente Sora.</p>

<!-- Image -->
<img src="image.jpg" class="img-fluid" alt="Descripción">

<!-- Table -->
<table class="table">
  <thead>
    <tr>
      <th>Columna 1</th>
      <th>Columna 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Dato 1</td>
      <td>Dato 2</td>
    </tr>
  </tbody>
</table>
```
